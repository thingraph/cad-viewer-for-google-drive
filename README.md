# CAD Viewer for Google Drive™

A web application that lets you view 2D and 3D CAD files from Google Drive™. In the Google Drive™ web interface, right‑click a supported file and choose **Open with** → **CAD Viewer** to open and view it in the browser.

## Supported File Formats

**2D CAD**

- **DWG** - AutoCAD Drawing files
- **DXF** - Drawing Exchange Format

**3D Models**

- **glTF / GLB** - GL Transmission Format (JSON and binary)
- **OBJ** - Wavefront Object
- **FBX** - Autodesk Filmbox
- **STL** - Stereolithography (3D mesh)

## Features

- 🎨 **2D CAD Viewer**: High-performance WebGL-based viewer for DWG/DXF files
- 🧊 **3D Model Viewer**: View glTF, GLB, OBJ, FBX, and STL 3D models in the browser
- 📏 **Measurement Tools**: Measure distance, area, angle, and coordinates
- ✏️ **Markup & Annotation**: Add arrows, rectangles, circles, text, and other annotations
- 📸 **Screenshot**: Capture and export high-quality images of drawings
- ⚙️ **Settings Panel**: Customize viewer settings and preferences
- 🎛️ **Interactive Toolbar**: Easy access to all viewer tools and features
- 🔄 **Layout Switching for 2D**: Switch between different layouts in multi-layout drawings
- 📑 **Layer Management for 2D**: Show/hide layers and manage layer visibility
- 🎯 **Navigation Tools**: Zoom, pan, rotate with axis gizmo and bottom bar controls

## Install

Install from **Google Workspace Marketplace** (recommended):

**[→ Install CAD Viewer from Google Workspace Marketplace](https://workspace.google.com/marketplace/app/dwg_viewer/641533811831)**

After installing, right‑click any supported CAD file (DWG, DXF, glTF, GLB, OBJ, FBX, STL) in Google Drive™ and choose **Open with** → **CAD Viewer** to use the app.
